= Установить сервис полнотекстового индексирования отдельно

Сервис полнотекстового индексирования при необходимости может быть установлен отдельно от сервера {dv} на любой компьютер (см. xref:ROOT:requirements-full-text-indexing.adoc[требования]) организации.

Отдельная установка сервиса полнотекстового индексирования позволяет снять часть нагрузки, связанной с индексированием данных, с сервера {dv}.

.Чтобы отдельно установить сервис полнотекстового индексирования:
. Установите серверную часть модуля {pl} на отдельный компьютер.
+
WARNING: В списке устанавливаемых компонентов модуля должны быть выбраны все компоненты.
+
.Набор компонентов модуля {pl}
image::admin:install-server-components.png[Набор компонентов модуля {pl}]
+
При завершении установки снимите флаг `*Запустить "Мастер настройки {dv}" после установки*`.
+
. xref:ROOT:requirements-full-text-account.adoc[Настройте] учетную запись сервиса полнотекстового поиска на данном компьютере.
. На компьютере с сервером {dv} откройте _{cns}_.
+
WARNING: Учетная запись, от имени которой запущена _{cns}_, должна иметь права на запись и чтение в ветку реестра `{hklm-dv}` на компьютере, куда устанавливается сервис полнотекстового поиска.
+
include::admin:partial$excerpts.adoc[tags=openPage]
+
.Настройки сервиса полнотекстового поиска
image::admin:fulltext-settings.png[Настройки сервиса полнотекстового поиска]
+
. Нажмите кнопку *…* в поле _Имя компьютера с установленным сервисом полнотекстового индексирования_, чтобы открыть окно для ввода названия компьютера.
. Введите полное сетевое имя компьютера, на который был установлен сервис полнотекстового поиска, например, `fulltext.company.com`. Затем нажмите кнопку *ОК*, чтобы записать название компьютера в текущее поле.
. Подключите БД к сервису полнотекстового индексирования xref:search-microsoft.adoc[Microsoft] или xref:search-elastic-configure.adoc[{es}].
